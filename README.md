# Arizona Prime — GitHub Pages

## Загрузка на GitHub

1. Создайте новый репозиторий на GitHub (например, `arizona-prime`)
2. В настройках репо: **Settings → Pages**
3. **Source:** Deploy from a branch
4. **Branch:** main
5. **Folder:** / (root)
6. Сохраните

## Файлы для загрузки

Загрузите в **корень** репозитория (ветка main):

- **index.html** — главный файл (обязательно)
- **404.html** — редирект для SPA (по желанию)

Через веб-интерфейс GitHub: перетащите файлы в область загрузки.
